hexo+github搭建静态博客及博客更新

关于搭建个人博客，好久之前就有这种想法，最近终于决定要搭建自己的博客。
在网上各种查找，终于找到了一个快速开发静态博客的框架。

hexo

 作为一个互联网从业者，相信好多人也和我一样，为了搭建自己的博客而苦恼；原因有很多，时间、精力、资源，相信没有一个人希望花费太多的时间和精力去维护管理。

经多方查询及朋友的经验，尝试着使用hexo框架来搭建我的个人博客。
